# same-tree

## 题目

给你两棵二叉树的根节点 p 和 q ，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

例 1：
![alt example1](/images/ex1)
>输入：p = [1,2,3], q = [1,2,3]
>输出：true


![alt example2](/images/ex2)
>输入：p = [1,2], q = [1,null,2]
>输出：false


![alt example3](/images/ex3)
>输入：p = [1,2,1], q = [1,1,2]
>输出：false
